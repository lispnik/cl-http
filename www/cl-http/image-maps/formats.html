Image-Map Formats

The following image-map formats are currently supported on CL-HTTP:

The CERN Format (:cern)
The image-map file format supported by the CERN Server versions 2.14 and later.

The NCSA Format (:ncsa)
The image-map file format supported by the NCSA HTTPd server.
