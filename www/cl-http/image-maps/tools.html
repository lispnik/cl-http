Image-Map Tools

Here is a listing of some of the available tools helpful in putting together image-maps. This list contains information about commercial, shareware, and freeware tools. It is by no means comprehensive. Whenever possible, a pointer is given to the original distributor of the software. For many shareware tools, pointers are given to other sites currently archiving it.

Macintosh

WebMap for the Macintosh Homepage
This is a shareware program (version 2.0 is commercial) for creating CERN and NCSA map files. WebMap 2.0b8 (226K) is available as shareware but has some major bugs. You can also download the older WebMap 1.0.1 (198K) from this site.

GIFConverter 2.3.7 (486K)
This is a good utility for basic cropping and conversion to GIF format, not an editor. INFO-MAC HyperArchive.

Imagery 1.9 Graphics Converter (216K)
This is a versitile, freeware conversion utility that handles graphics of numerous formats and platforms. INFO-MAC HyperArchive.

Transparency 1.0 (37K)
A basic utility that sets the transparency index of GIF images. This allows "borderless" images on most web browsers. INFO-MAC HyperArchive.

Jon Wiederspan's Macintosh WWW Development Guide
This site contains a number of good utilities for creating imagemaps and converting graphics.

Unix

Mapedit for X 1.1.2 Homepage
This is a WYSIWYG map editor for UNIX that you will need to compile. Download SunSITE.

NETPBM Graphics Conversion Package (825 K)
This conversion utility handles both GIF transparency and conversion. X Consortium.

Windows

Mapedit 1.4 (57 K)
This map editor is by the author of Mapedit for X. SunSITE.
